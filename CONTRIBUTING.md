# Contributing

This is a **portfolio and architecture demonstration** repository. The full implementation of the ARE Audit Pipeline is private to ensure client confidentiality.

## How to Contribute
While we are not accepting major features for this specific demo, we welcome:
*   Discussions on **GDPR-compliant AI architectures**.
*   Improvements to the **Rust/Python interoperability** examples.
*   Feedback on **geospatial compliance systems**.

## Issues and Discussions
If you have questions about the design decisions documented in the [ADR](./adr/) folder, please open an issue or start a discussion.

## Pull Requests
If you submit a pull request, please ensure it aligns with the core architectural principles documented in [ARCHITECTURE.md](./ARCHITECTURE.md).
